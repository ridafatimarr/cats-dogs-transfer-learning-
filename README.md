# Cats and Dogs Image Classification using Transfer Learning (VGG16, VGG19, ResNet50)
Transfer learning-based image classification using VGG16, VGG19, and ResNet50 to classify cats and dogs. Covers preprocessing, augmentation, fine-tuning, and comparative analysis of model performance with improved accuracy and reduced overfitting. Framework: TensorFlow / Keras
# Results
VGG16: 91 % accuracy  (best performance) 
VGG19: 87 % accuracy  
ResNet50: 62 % accuracy  
ResNet50 showed lower performance compared to VGG-based models, possibly due to  dataset size limitations.Further optimization (e.g., deeper fine-tuning or hyperparameter tuning) could improve its results.
